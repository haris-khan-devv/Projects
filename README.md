## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/portfolio-projects.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd portfolio-projects
   ```

3. **Open any HTML file in your browser**
   - Simply double-click any `.html` file
   - Or use a local server (VS Code Live Server, Python HTTP server, etc.)

4. **No build process required!** These are pure HTML/CSS/JS projects
